### 关键发现

1. **南部省份优势明显**：南部工业省份年均 GHI 为 1,752~1,875 kWh/m²，比北部高约 **28%~35%**；光伏发电潜力比北部高约 **25%~32%**。
2. **巴地头顿省最优**：在南部工业省份中，巴地头顿省（沿海，富美工业区所在地）太阳能资源最好，GHI 达 1,875 kWh/m²/年，PVOUT 达 1,482 kWh/kWp/年。
3. **胡志明市次之**：GHI 1,841 kWh/m²/年，PVOUT 1,455 kWh/kWp/年，作为经济中心具有良好的屋顶光伏潜力。
4. **隆安省相对较低**：在南部省份中 GHI 最低（1,752），可能与地处湄公河三角洲、雨季湿度较高有关。
5. **中部参考省份最优**：宁顺省是全越南太阳能资源最好的省份（年均日照 2,843 小时），但距离主要工业区较远，主要适合大型地面电站。
6. **北部省份可行但回报较低**：北部工业区（北宁、北江等）PVOUT 约 1,100~1,160 kWh/kWp/年，虽然低于南部，但对于工厂屋顶自用项目仍具经济可行性。

## 数据来源与方法说明

### 主要数据来源

1. **Global Solar Atlas / Solargis**（globalsolaratlas.info）— 世界银行支持的全球太阳能数据平台，基于 Solargis 卫星遥感模型，空间分辨率 250m（GHI）/ 1km（PVOUT）
2. **NASA POWER / HOMER**（power.larc.nasa.gov）— NASA 地表太阳辐射数据库，为河内和胡志明市提供了实测级日均 GHI 数据
3. **profileSOLAR.com** — 基于 NASA POWER API 的光伏分析工具，提供了最优倾角和季节性输出数据
4. **越南工贸部（MOIT）三区分类**（Decision No. 13/2020/QD-TTg）— 将越南 63 省分为三个辐照区，为数据验证提供了制度性参考
5. **越南国家气象水文局（VNMHA）** — 宁顺省日照数据的官方来源

### 数据推导方法

- **GHI**：河内和胡志明市采用 NASA POWER（HOMER）实测卫星数据；其他省份基于 Global Solar Atlas GIS 数据层级和 MOIT 三区分类进行插值，并与 Solargis 地图进行交叉验证
- **日照小时数（PSH）**：PSH = 年均 GHI ÷ 365（1 峰值日照小时 = 1 kWh/m² 的辐照量）
- **最优倾角**：基于 profileSOLAR.com 对越南 27 个位置的分析结果。南部省份（纬度 ~10-11°N）最优倾角约 10°，北部省份（纬度 ~21°N）约 17-18°
- **PVOUT**：基于 GHI 数据，考虑最优倾角的辐照增益（GTI/GHI 比率约 1.04~1.08）和系统总损耗（温度损耗 + 逆变器 + 线缆 + 积灰，南方约 21~24%，北方约 18~21%），推算公式为 PVOUT ≈ GHI × 倾角增益 × (1 - 系统损耗)

### 数据精度说明

- 本数据为**省级/城市级参考值**，实际项目应使用 Global Solar Atlas 或 Solargis 的精确坐标查询功能获取站点级数据
- GHI 数据精度约 ±5~8%（Solargis 验证标准偏差），PVOUT 精度约 ±8~12%
- 实际光伏系统发电量还受组件类型、安装方式（屋顶朝向/倾角）、积灰、遮挡、电网可用性等因素影响
- 北宁省三星项目的实测数据（2.38 MWp → 259 万 kWh/年 → ~1,088 kWh/kWp/年）可作为北部实际发电量的参考基准

### 验证说明

数据已通过以下方式交叉验证：
1. 所有南部省份 GHI 值落在 MOIT Region 3 范围内（1,704~1,910 kWh/m²/年）✓
2. 所有北部省份 GHI 值落在 MOIT Region 1 范围内（1,226~1,433 kWh/m²/年）✓
3. 南部 > 北部的趋势一致 ✓
4. 宁顺省 GHI 为全表最高值 ✓
5. 巴地头顿省在南部工业省份中最优（与文献一致）✓
6. PVOUT 值落在 Global Solar Atlas 越南全国统计的合理百分位范围内 ✓
